# 🏠 Housing Data Preprocessing
Task 1

## 📌 Overview
This project performs data cleaning and preprocessing on a housing dataset to prepare it for machine learning models.

---

## 🛠️ Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

## 🔄 Steps Performed
- Imported and explored dataset
- Handled missing values using mean imputation
- Encoded categorical variables using Label Encoding
- Detected and removed outliers using IQR method
- Standardized numerical features using StandardScaler
- Saved cleaned dataset for ML use

---

## 📊 Visualizations
- Boxplots used for outlier detection before and after cleaning

---

## 📁 Output
- Cleaned dataset saved as `preprocessed_data.csv`
